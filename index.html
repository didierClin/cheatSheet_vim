l'éditeur modal ultra-puissant !

Les trois modes principaux de vi :
  • le mode normalC'est le mode par defaut à l'ouverture de vim. Il permet de se déplacer dans le fichier.
  • Le mode commande: Dans ce mode on donne des instructions à Vim. Editer, sauvegarder, quitter,..
  • Le mode insertion: permet d'inserer le texte et son formatage.

le mode normal :

C'est le mode par defaut à l'ouverture de vim.
La touche 'echap' nous ramene dans ce mode. Naviguer dans le texte:
  • h, l : déplacement respectif à gauche ou a droite,
  • j, k : déplacement respectif en haut ou vers le bas,
    nb: les touches fleches haut, bas, gauche et droite ont les memes actions.
  • gg : va au debut du texte, G : va à la fin du texte,
  • 0 : va au debut de la ligne, $ : va à la fin de la ligne,
  • { : va au début du paragraphe, } : va à la fin du paragraphe,
  • w : avance d'un mot, b : recule d'un mot
  • e : avance jusqu'à la fin d'un mot,
Effacement:
  • x : supprime le caractère sous le curseur
  • X : supprime le caractère précedent le curseur,
  • d : supprime ..
  • dw : supprime le mot sous le cursuer,
  • d$ : supprime depuis le curseur jusqu'en fin de ligne,

le mode commande :

Pour saisir une commande, taper ':' suivi d'une commande. les touches principales:
  • w : sauvegarde du fichier
  • q : quitter le fichier sans sauvegarde, une alerte apparait si le fichier est modifié
  • q! : forcer la sortie du fichier sans sauvegarde, evite une alerte
  • wq ou x : enregistrer le fichier et sortir de vim,
  • e mon_fich : éditer le fichier mon_fich, mon_fich sera créé si il n'existe pas,
